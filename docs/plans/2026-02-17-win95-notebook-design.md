# Windows 95 Notebook Portfolio — Design

## Summary

Replace the current academic website with a full Windows 95 desktop simulation that renders a Jupyter notebook as the main content. The notebook is the source of truth for resume content — edit the `.ipynb`, push, and the site auto-updates.

## Architecture

### Desktop Layer
- Full-viewport `#008080` teal background (classic Win95 desktop)
- Desktop icons in a left-column grid:
  - `My Resume.ipynb` — opens/focuses the notebook window
  - `Email.exe` — mailto link
  - `GitHub.lnk` — opens GitHub profile
  - `LinkedIn.lnk` — opens LinkedIn profile
  - `CV.pdf` — opens resume PDF
- Double-click to activate icons

### Notebook Window
- 98.css `window` class for authentic Win95 chrome
- Title bar: "Jerry Han - Resume.ipynb - Jupyter Notebook" with min/max/close buttons
- Decorative menu bar (File / Edit / View)
- Body: converted `.ipynb` HTML with Jupyter-style cells (`In[n]:` prompts, code/markdown distinction)
- Draggable via title bar, minimizable to taskbar, maximizable, closable (reopenable from icon)

### Taskbar
- Fixed at viewport bottom
- Start button (decorative)
- Active window entry for the notebook
- Real-time clock

## Tech Stack

- **98.css** — CSS-only Win95/98 UI library (~10KB, no JS)
- **Custom win95.js** — window dragging, minimize/maximize/close, taskbar clock, icon handlers
- **Custom win95.css** — desktop layout, icon grid, notebook cell overrides
- **GitHub Action** — runs `jupyter nbconvert --to html --template basic` on push
- **No build system** — remains a static site on GitHub Pages

## File Structure

```
website/
├── index.html                          — Win95 desktop shell, embeds notebook.html
├── win95.css                           — Desktop, taskbar, icon, notebook cell styles
├── win95.js                            — Window management, clock, icon interactions
├── resume.ipynb                        — Notebook (content source of truth)
├── notebook.html                       — Auto-generated by GitHub Action
├── .github/workflows/convert-notebook.yml
├── images/                             — Existing + Win95 desktop icons
└── (existing files preserved)
```

## GitHub Action Pipeline

1. Triggers on push when `resume.ipynb` changes
2. Installs Python + Jupyter
3. Runs `jupyter nbconvert --to html --template basic resume.ipynb`
4. Commits generated `notebook.html` back to repo
5. GitHub Pages auto-deploys

## Notebook Styling

- Preserve Jupyter cell structure (In[n] prompts, cell borders)
- Code cells: monospace, light syntax highlighting
- Markdown cells: rendered headings, lists, links, images
- Slightly retro font feel while remaining readable

## Interactivity

- Window drag via title bar
- Minimize collapses to taskbar, maximize fills viewport minus taskbar
- Close hides window, reopen from desktop icon double-click
- Taskbar clock shows real time
- Desktop icons: double-click to activate
